Comprendre et maîtriser le CSS

Le CSS (Cascading Style Sheets) permet de styliser vos pages web pour les rendre visuellement attrayantes.

Quelques propriétés essentielles :

  • color : modifie la couleur du texte
  • background-color : change la couleur de fond
  • padding : espace intérieur d'un élément

Exemple de code :

h1 {
    color: #6B9080;
    font-size: 36px;
    text-align: center;
}
